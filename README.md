🎉 Event Manager SPA
Une application de gestion d’événements construite en Single Page Application (SPA) avec HTML, CSS et JavaScript vanilla. Elle permet d’ajouter, visualiser, trier et gérer des événements dynamiquement, sans rechargement de page.

🚀 Fonctionnalités principales
📊 Dashboard / Stats
Affichage du nombre total d’événements

Calcul du nombre total de places

Somme théorique des revenus (prix × places)

(Bonus) : Affichage d’un graphe des statistiques

➕ Add Event
Formulaire dynamique pour ajouter un événement

Gestion des variantes (ajout/suppression de sous-événements)

Validation des champs et affichage des erreurs

📋 Events List
Tableau dynamique des événements

Recherche par titre

Tri par titre ou prix

Actions :

Voir les détails (modal)

Éditer un événement (pré-remplit le formulaire)

Supprimer (archivage logique)

(Bonus) : Pagination (5 à 6 événements par page)
